js dom merupakan mempresentasikan element html menjadi dom tree yang mempunyai node di dalamnya dan 1 node itu mempresentasikan masing masing element html,text dan lain lain yang berbentuk object yang dapat kita manipulasi 
    -root node merupakan sumber/akar dari node node yang ada secara default root node itu adalah document akar nya tetap tag <html>
    -nodelist merupakan kumpulan banyak node node campur seperti element html,text, dan lain lain
    -htmlCollection merupakan kumpulan banyak node node yang hanya berjenis element html saja
htmlCollection bersifat live saat dom berubah akan ikut langsung berubah dan jika nodelist bisa bersifat static ataupun live tergantung tetapi keduanya masih tetap bisa di manipulasi
